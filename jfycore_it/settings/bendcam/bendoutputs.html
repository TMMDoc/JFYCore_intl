Uscite di piega

Report Settings

Report template - È la posizione e il nome del modello da utilizzare per la comparsa dei report.

Report destination - Impostare come posizione dove salvare tutti i file dei report.

Report output format - Può essere impostato come HTML o PDF. Nota: affinché il PDF utilizzi il contenuto multimediale e 3D, occorre installare Adobe Acrobat Reader.

Generate report on switching to bend - Attivare affinché un report venga generato automaticamente quando si passa a una piega ed è stata trovata una soluzione. Se la soluzione rileva errori o avvertimenti, il report non verrà generato.

Attach reports to an NC program if possible - Attivare questa funzione per allegare report al programma NC.

Code Settings

NC-Code destination - Impostare come posizione dove salvare tutti i file di programma NC.

NC-Code post-processor - È la posizione e il nome di un file batch o di un utensile di altro produttore a cui fare riferimento quando si genera il programma NC.

Generate code on switching to bend - Attivare affinché il programma NC venga generato automaticamente quando si passa alla modalità Bend CAM (a condizione che venga generata una soluzione valida senza errori).

Simulation graphics in NC code - È un’opzione B23 che permette di confrontare la grafica BNC-SAT con il pezzo effettivo. La grafica BNC-Sat viene visualizzata su un monitor. Un’immagine del pezzo effettivo viene visualizzata sullo stesso monitor. L’utente può ora spostare, ruotare o girare il pezzo fino a quando la grafica BNC e le immagini del pezzo effettivo si sovrappongono. Questo aiuta l’utente a vedere se il pezzo è posizionato correttamente per la piegatura.

Simplify graphics (remove small holes) - Per impostazione predefinita, è attivato (e la grafica BNC è semplificata). Può però essere disattivato; se disattivato, tutti i fori vengono emessi in BNC.

Lock code (no recalculation by HMI) - Attivare questa opzione per bloccare il codice. Ciò fa sì che il controller non modifichi il codice né esegua ricalcoli.

Format for flat pattern - Può essere impostato come GEO, DXF, o PDG per l’opzione di esportazione dei file predefinita quando si salva automaticamente un design piatto. Quando viene trovata una soluzione per un pezzo, il modello piatto può automaticamente essere esportato e salvato nella destinazione del modello piatto. Tuttavia, se si trovano errori o avvertimenti, il design piatto dovrà essere salvato manualmente.

Flat pattern destination - Impostare come posizione dove salvare tutti i modelli piatti.

Generate FX file along with NC code - Attivare questa opzione affinché un file FX venga salvato nella posizione scelta durante la generazione del programma NC.

FX file destination - Impostare come posizione per il salvataggio di tutti i file FX.
