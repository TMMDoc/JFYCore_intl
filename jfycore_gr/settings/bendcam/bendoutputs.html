Αποτελέσματα κάμψης

Report Settings

Report template - Αυτή είναι η τοποθεσία και το όνομα του προτύπου που θα χρησιμοποιηθεί για την εμφάνιση των αναφορών.

Report destination - Ορίστε αυτήν ως την τοποθεσία όπου πρέπει να αποθηκεύονται όλα τα αρχεία αναφοράς.

Report output format - Αυτό μπορεί να οριστεί είτε ως HTML είτε ως PDF. Σημείωση: προκειμένου η μορφή PDF να χρησιμοποιεί το περιεχόμενο πολυμέσων και 3D πρέπει να εγκατασταθεί το πρόγραμμα Adobe Acrobat Reader.

Generate report on switching to bend - Ενεργοποιήστε αυτήν την επιλογή προκειμένου να δημιουργείται αυτόματα μια αναφορά όταν μεταβαίνετε στην κάμψη και έχει βρεθεί μια λύση. Εάν η λύση ανιχνεύσει σφάλματα ή προειδοποιήσεις, η αναφορά δεν θα δημιουργηθεί.

Attach reports to an NC program if possible - Ενεργοποιήστε αυτήν τη λειτουργία για να επισυνάψετε αναφορές στο πρόγραμμα NC.

Code Settings

NC-Code destination - Ορίστε αυτήν ως την τοποθεσία για όλα τα αρχεία προγράμματος NC που πρέπει να αποθηκευτούν.

NC-Code post-processor - Αυτή είναι η τοποθεσία και όνομα ενός αρχείου δέσμης ή εξωτερικού εργαλείου που πρέπει να αναφέρεται κατά τη δημιουργία του προγράμματος NC.

Generate code on switching to bend - Ενεργοποιήστε αυτήν την επιλογή για το πρόγραμμα NC να δημιουργείται αυτόματα κατά τη μετάβαση στη λειτουργία Bend CAM (υπό την προϋπόθεση ότι δημιουργείται μια έγκυρη λύση χωρίς σφάλματα).

Simulation graphics in NC code - Αυτή είναι μια επιλογή B23 που σάς παρέχει τη δυνατότητα να συγκρίνετε τα γραφήματα BNC-SAT με το πραγματικό τεμάχιο. Το γράφημα BNC-Sat προβάλλεται σε μια οθόνη. Μια εικόνα του πραγματικού τεμαχίου προβάλλεται στην ίδια οθόνη. Ο χρήστης μπορεί τώρα να μετακινήσει, να περιστρέψει ή να γυρίσει το τεμάχιο μέχρι το σημείο όπου τα γραφήματα BNC και οι εικόνες του πραγματικού τεμαχίου αλληλεπικαλύπτονται. Αυτό βοηθά τον χρήστη να εξακριβώσει αν το τεμάχιο είναι τοποθετημένο σωστά για κάμψη.

Simplify graphics (remove small holes) - Από προεπιλογή, αυτή η λειτουργία είναι ενεργοποιημένη (και τα γραφήματα BNC είναι απλουστευμένα). Ωστόσο, αυτή η λειτουργία μπορεί να απενεργοποιηθεί και τότε όλες οι οπές εξάγονται στο BNC.

Lock code (no recalculation by HMI) - Ενεργοποιήστε αυτήν την επιλογή για να κλειδώσετε τον κωδικό. Έτσι διασφαλίζεται ότι ο ελεγκτής δεν αλλάζει τον κωδικό ούτε διεξάγει οποιονδήποτε επανυπολογισμό.

Format for flat pattern - Αυτό μπορεί να οριστεί ως GEO, DXF, ή PDG για την προεπιλεγμένη επιλογή εξαγωγής αρχείου κατά την αυτόματη αποθήκευση ενός επίπεδου σχεδίου. Όταν βρεθεί μια λύση για ένα τεμάχιο, το επίπεδο μοτίβο μπορεί να εξαχθεί και να αποθηκευτεί αυτόματα στον προορισμό επίπεδου μοτίβου, ωστόσο, εάν διαπιστωθούν σφάλματα ή προειδοποιήσεις, το επίπεδο σχέδιο θα πρέπει να αποθηκευτεί χειροκίνητα.

Flat pattern destination - Ορίστε αυτήν ως τοποθεσία. αποθήκευσης όλων των επίπεδων μοτίβων.

Generate FX file along with NC code - Ενεργοποιήστε αυτήν την επιλογή για να αποθηκεύσετε ένα αρχείο FX στην επιλεγμένη τοποθεσία κατά τη δημιουργία του προγράμματος NC.

FX file destination - Ορίστε αυτήν ως την τοποθεσία για την αποθήκευση όλων των αρχείων FX.
