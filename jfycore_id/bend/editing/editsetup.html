Edit pengaturan penekukan

Saat mengedit pemasangan penekukan, Anda dapat mengklik tombol navigasi Setup di panel pemasangan alat untuk mengedit seluruh pengaturan penekukan. Panel Pengaturan Penekukan ditampilkan di samping, dan ada beberapa aktivitas yang dapat dilakukan di sini.

  • Anda dapat Save pengaturan ke file pengaturan penekukan (file ini memiliki ekstensi .bsetup). Nantinya, Anda dapat menggunakan opsi Load…​ untuk memuat pengaturan ini untuk digunakan dengan komponen yang berbeda. Ini adalah cara manual sederhana untuk menggunakan pengaturan yang sama untuk beberapa komponen.

  • Tombol Merge Setups muncul jika ada beberapa pengaturan dalam komponen, dan tombol ini menggabungkan beberapa pengaturan tersebut menjadi satu, jika ada cukup jarak bebas yang tersedia pada mesin untuk semua setups[1].

  • Tombol Center Setup akan muncul jika stasiun-stasiun dalam pengaturan ini tidak berada di tengah area meja yang tersedia pada mesin.

  • Tombol Prev dan Next dapat digunakan untuk beralih di antara pengaturan yang berbeda dalam komponen ini (jika ada beberapa pengaturan).

  • Mengklik tombol navigasi Mounts akan membawa Anda satu tingkat ke bawah, ke pengeditan pemasangan alat pertama dalam pengaturan ini (ini adalah kebalikan dari tombol navigasi Setup di panel pemasangan alat).

1. Jika pengaturan menggunakan alat dengan ketinggian yang berbeda, hal ini sering kali akan menyebabkan benturan antara punch dan die, yang akan dilaporkan dalam Navigator Penekukan.
