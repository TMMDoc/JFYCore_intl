Καρτέλα μεταλλικών πλακών

Η καρτέλα sheets του πίνακα εργασίας χρησιμοποιείται για την προσθήκη πρώτης ύλης (μεταλλικών πλακών) για κατανομή.

  • Χρησιμοποιήστε το Add για να προσθέσετε μια ορθογώνια μεταλλική πλάκα.

    • Το Size καθορίζει το πλάτος και το ύψος της μεταλλικής πλάκας και το Margins καθορίζει τα περιθώρια στις τέσσερις πλευρές όπου δεν πρέπει να τοποθετούνται τεμάχια.

    • Το Quantity είναι ο αριθμός των μεταλλικών πλακών που είναι διαθέσιμες στο απόθεμα.

    • Το Priority χρησιμοποιείται για να προτείνει στη μηχανή κατανομής τις μεταλλικές πλάκες που θα πρέπει να επιχειρήσει να χρησιμοποιήσει πρώτα (αρχίζοντας από το μικρότερο αριθμό).

    • Το Rolling direction χρησιμοποιείται για τον καθορισμό του κόκκου της μεταλλικής πλάκας και είναι χρήσιμο κατά την κατανομή των τεμαχίων που έχουν μια συγκεκριμένη προδιαγραφή κόκκων - το JFY Cut ταυτίζει τον κόκκο περιστρέφοντας κατάλληλα το τεμάχιο κατά την κατανομή.

  • Χρησιμοποιήστε το Import για την εισαγωγή μιας ακανόνιστης μεταλλικής πλάκας (μια υπολειπόμενη μεταλλική πλάκα από προηγούμενη κατανομή ή που έχει δημιουργηθεί από άλλο λογισμικό). Η μεταλλική πλάκα μπορεί να είναι οποιαδήποτε μορφή 2D, όπως ένα αρχείο .DXF, .GEO ή .FX. Θα πρέπει να υπάρχει ένα ενιαίο κλειστό εξωτερικό περίγραμμα που είναι το όριο της μεταλλικής πλάκας. Τυχόν κλειστά εσωτερικά περιγράμματα μέσα σε αυτό ερμηνεύονται ως οπές στις οποίες δεν πρέπει να τοποθετούνται τεμάχια. Μπορείτε να δείτε μια τέτοια εισαγόμενη ακανόνιστη μεταλλική πλάκα στην παραπάνω εικόνα.

  • Χρησιμοποιήστε το Delete για να διαγράψετε μια μεταλλική πλάκα από το απόθεμα.
