Αναφορές

Εκτός από τη δημιουργία προγραμμάτων NC, το JFY Bend μπορεί επίσης να δημιουργήσει διάφορες αναφορές για ένα τεμάχιο, ανάλογα με τις μονάδες που έχουν εγκατασταθεί. Όταν εργάζεστε με την τεχνολογία κάμψης, για παράδειγμα, είναι χρήσιμο να δημιουργήσετε ένα φύλλο ρυθμίσεων που παραθέτει τα εργαλεία που πρέπει να εγκατασταθούν και παρέχει λεπτομέρειες για την αλληλουχία κάμψης.

Το πρόγραμμα Report Designer (Σχεδιαστής αναφορών) στο JFY Bend σάς βοηθά να σχεδιάσετε το περιεχόμενο και το στιλ τέτοιων αναφορών. Αφού σχεδιάσετε μια αναφορά, μπορείτε να την εξάγετε σε μορφή PDF ή HTML.[1] μορφές. Εδώ παρατίθεται η εμφάνιση του σχεδιαστή αναφορών όταν είναι ανοικτός:

Χρήση μιας αναφοράς

Μπορείτε να ανοίξετε μια αναφορά για επεξεργασία επιλέγοντας το μενού Αρχείο/Άνοιγμα και επιλέγοντας ένα αρχείο προτύπου αναφοράς (αυτά είναι αρχεία με την επέκταση .ftrep). Μπορείτε να βρείτε μερικά δείγματα προτύπων αναφοράς στο φάκελο Samples (Δείγματα) της εγκατάστασής JFY Bend.

Προτού να είστε σε θέση να ανοίξετε ένα πρότυπο αναφοράς για να εργαστείτε με αυτό, θα πρέπει πρώτα να έχετε ανοίξει ένα τεμάχιο και να έχετε ολοκληρώσει την επεξεργασία του τεμαχίου (θα πρέπει να δημιουργήσετε τα κατάλληλα δεδομένα τεχνολογίας Κάμψης/Κοπής/Δίπλωσης). Αυτό είναι αναγκαίο επειδή ο σχεδιαστής αναφορών λειτουργεί με δεδομένα σε πραγματικό χρόνο. Προβάλλει επίκαιρα δεδομένα από το τεμάχιο καθώς σχεδιάζετε την αναφορά, έτσι ώστε να μπορείτε να δείτε άμεσα ποιο θα είναι το τελικό εκτυπωμένο αποτέλεσμα.

Μπορείτε επίσης να ξεκινήσετε με μια κενή αναφορά επιλέγοντας το μενού Αρχείο/Νέο/Πρότυπο αναφοράς. Στη συνέχεια, μπορείτε να συμπληρώσετε την αναφορά προσθέτοντας διάφορα στοιχεία αναφοράς (πεδία κειμένου, εικόνες, ετικέτες κ.λπ.) και στη συνέχεια το αποθηκεύετε.

Όταν έχετε ανοιχτό ένα πρότυπο αναφοράς, μπορείτε να χρησιμοποιήσετε την εντολή Αρχείο/Εξαγωγή σε PDF ή Αρχείο/Εξαγωγή σε HTML για να διαμορφώσετε την αναφορά σε μορφή PDF ή σε μορφή HTML. Η αναφορά εκτυπώνεται χρησιμοποιώντας τα δεδομένα από το τεμάχιο που είναι ανοικτό και ενεργό επί του παρόντος. Μπορείτε επίσης να αλλάξετε την αναφορά μεταξύ της προβολής δεδομένων σε πραγματικό χρόνο (η προεπιλογή) και μιας προβολής σχεδίασης που εμφανίζει τις υποκείμενες εκφράσεις που χρησιμοποιούνται για τη συμπλήρωση των διαφόρων πεδίων στην αναφορά.

Αυτόματα εκτυπωνόμενες αναφορές:

Μπορείτε να ρυθμίσετε το JFY Bend ώστε να εκτυπώνει αυτόματα μια αναφορά κάμψης κάθε φορά που δημιουργείται μια λύση κάμψης (ακριβώς όπως αυτό μπορεί να δημιουργήσει αυτόματα ένα πρόγραμμα NC όταν υπολογίζεται η λύση). Χρησιμοποιήστε τη σελίδα ρυθμίσεων αποτελεσμάτων από τη μονάδα τεχνολογίας κάμψης για να διαμορφώσετε αυτήν την επιλογή.

Στο παραπάνω παράδειγμα ενεργοποιείται η αυτόματη δημιουργία αναφορών. Το πρότυπο αναφοράς έχει οριστεί σε lib:Report/bend.ftrep που είναι το προεπιλεγμένο πρότυπο αναφοράς για κάμψη CAM (αυτό εγκαθίσταται στο πλαίσιο κάθε εγκατάστασης JFY Bend). Εάν δημιουργήσετε ένα διαφορετικό πρότυπο αναφοράς χρησιμοποιώντας το σχεδιαστή αναφορών, μπορείτε να καθορίσετε εδώ αυτό το πρότυπο.

Οι ρυθμίσεις σάς παρέχουν επίσης τη δυνατότητα να ελέγχετε το φάκελο στον οποίο δημιουργείται η αναφορά και τη μορφή (HTML ή PDF) στην οποία εγγράφεται η αναφορά.

1. Τα αρχεία HTML που δημιουργούνται από συντάκτη της αναφοράς JFY Bend είναι ολοκληρωμένα. Περιλαμβάνουν όλες τις αναγκαίες εικόνες, τα φύλλα στιλ και άλλους πόρους ενσωματωμένα στο αρχείο HTML. Έτσι, μπορούν εύκολα να μεταφερθούν ή να αρχειοθετηθούν χωρίς εξωτερικές εξαρτήσεις.
