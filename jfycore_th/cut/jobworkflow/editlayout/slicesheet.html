แผ่นตัด

หากแผ่นโลหะมีการวางซ้อนบางส่วน คุณสามารถเพิ่ม SliceLine การตัดเพื่อเล็มส่วนสี่เหลี่ยมที่ไม่ได้ใช้ และสามารถใช้เป็นวัตถุดิบสำหรับงานซ้อนแผ่นในอนาคต หน้า Skeleton Cuts ของ การตั้งค่า JFY Core สามารถใช้กำหนดการสร้าง การตัดแผ่นอัตโนมัติได้ หากเหลือเศษแผ่นมากพอ

การตัดแผ่นจากแผง Layout

คุณยังสามารถเพิ่มการตัดแผ่นได้เองจากแผง Layout หากต้องการเปิดแผงนี้ ให้คลิกใกล้ขอบของแผ่น หรือ คลิกที่ส่วนใดก็ได้เพื่อเปิดแผง Placement แล้วคลิกที่ ปุ่มนำทาง Layout

ภาพด้านบนแสดงขั้นตอนการเพิ่มการตัดแผ่นด้วยตนเอง

  • คลิกใกล้ขอบเพื่อเปิดแผง Layout

  • คลิกปุ่ม Insert sheet cut เพื่อสลับไปยังโหมด ตัดแผ่น

  • คลิกที่ขอบของแผ่น แล้วใช้ snap แนวนอนหรือแนวตั้ง เพื่อวาดเส้นแนวนอนหรือแนวตั้งที่แม่นยำ คุณไม่จำเป็นต้องขยายเส้นนี้ ไปจนถึงขอบอีกด้านของแผ่น เมื่อคลิกอีกครั้ง เส้นจะถูก ขยายโดยอัตโนมัติให้เป็นการตัดเต็มแผ่นจากปลายถึงปลาย

คุณสามารถใช้ฟังก์ชันเดียวกันนี้เพื่อเพิ่ม Skeleton cuts ได้เช่นกัน เพียงวาดเส้น แล้วจะถูกแบ่งออกเป็นหลายส่วนโดยอัตโนมัติ ทำการตัดเฉพาะโครงของ แผ่นที่เหลือหลังจากที่ชิ้นส่วนทั้งหมดถูกตัดออกแล้ว:

การแก้ไขการตัดแผ่น

คุณสามารถคลิกที่เส้นการตัดเพื่อเปิดตัวแก้ไข Slice Line:

  • คลิกที่เส้นการตัดแล้วลากเพื่อเปลี่ยนตำแหน่ง คุณยังสามารถลากเส้นไปไว้หลังชิ้นส่วน และขอบเขตของเส้นร่วมกับชิ้นส่วนบนเลย์เอาต์จะถูกคำนวณใหม่ โดยอัตโนมัติ

  • คลิก Flip direction พื่อสลับทิศทางหรือ ลบ เพื่อลบเส้นการตัดออก

  • คลิก ส่วนแยกย่อย เพื่อแก้ไขรายละเอียดเพิ่มเติมของ เส้นการตัด

  • ใช้ Condition และ Pierce เพื่อเลือกเงื่อนไขการตัดและ ประเภทของการเจาะ

  • เมื่อเริ่มหรือตัดใกล้ขอบของแผ่น สิ่งสำคัญคือ ต้องจัดการเซ็นเซอร์วัดความสูงของหัวเลเซอร์ โดยทั่วไป การวัด จะทำที่ตำแหน่งห่างจากขอบแผ่นเพียงไม่กี่มิลลิเมตร (แสดงด้วยกากบาทสีแดง ในภาพด้านบน) และระบุด้วยพารามิเตอร์ Measure จากนั้น ความสูงของหัวเลเซอร์จะถูกล็อกไว้ ซึ่งจะเคลื่อนไปยังตำแหน่งเจาะ แล้วกลับมาที่ตำแหน่งวัดโดยไม่ใช้เซ็นเซอร์วัดความสูง ก่อนที่จะดำเนินการตัดต่อไป

  • การตั้งค่า Overtravel กำหนดระยะ ที่เส้นตัดแผ่นยื่นออกไปเหนือขอบแผ่นด้านบนและด้านล่าง

  • การตั้งค่า Sheet web จะสร้าง microjoint ที่มีขนาดที่กำหนดไว้ที่ด้านบนและด้านล่างของแผ่น

  • Part web คือ microjoint ที่เกิดขึ้นระหว่างเส้นตัดและขอบของชิ้นส่วนที่มีอยู่แล้ว

  • Pierce distance คือระยะห่างของ จุดเจาะจากขอบของชิ้นส่วน การเจาะเกิดขึ้นที่ระยะนี้จากชิ้นส่วน จากนั้นหัวฉีดจะเคลื่อนเข้าใกล้ชิ้นส่วนจนถึง ระยะ Part web ที่กำหนด แล้วหมุน 180° เพื่อดำเนินการตัดต่อ
