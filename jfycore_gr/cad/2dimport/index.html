Εισαγωγή 2-D

Το JFY Core μπορεί να εισάγει σχέδια 2-D από αρχεία DXF ή GEO. Αυτά τα σχέδια είναι συνήθως επίπεδα 2-D - είναι αναδιπλώσεις ή αναπτύξεις ενός τεμαχίου μεταλλικής πλάκας 3-D που είναι έτοιμο για επεξεργασία με μηχάνημα κοπής λέιζερ.

Για την επεξεργασία για λέιζερ, καμπτικές πρέσες ή διπλωτικές πρέσες πάνελ, το JFY Core χρειάζεται ένα σχέδιο μεταλλικής πλάκας που περιλαμβάνει τα εξής:

  • Ένα κλειστό εξωτερικό περίγραμμα

  • Ένα ή περισσότερα εσωτερικά περιγράμματα (οπές)

  • Μία ή περισσότερες γραμμές κάμψης

Ανατρέξτε στη σελίδα Εκκαθάριση 2-D για περισσότερες πληροφορίες σχετικές με τον τρόπο χρήσης των εργαλείων λειτουργίας σχεδίασης για την εκκαθάριση ενός σχεδίου ή για την προσθήκη σχολιασμών γραμμής κάμψης. Όταν θα είναι έτοιμο το σχέδιο, μπορεί να χρησιμοποιηθεί για περαιτέρω επεξεργασία:

  • Πατήστε 3 για να διπλώσετε το σχέδιο σε ένα μοντέλο 3-D (χρησιμοποιώντας τις σημειώσεις γραμμής κάμψης που υπάρχουν στο σχέδιο).

  • Πατήστε L για να επεξεργαστείτε το σχέδιο για κοπή με λέιζερ.

  • Πατήστε B για να επεξεργαστείτε το σχέδιο για κάμψη με μήτρα σε μια καμπτική πρέσα

  • Πατήστε F για να επεξεργαστείτε το σχέδιο σε μια διπλωτική μηχανή πάνελ

  • Πατήστε W για να ανοίξετε τον πίνακα Ροής Εργασιών να πραγματοποιήσετε όλη την επεξεργασία αυτού με πιο διαδραστικό τρόπο.

Μπορείτε επίσης να δημιουργήσετε ένα σχέδιο 2-D εντελώς από την αρχή χρησιμοποιώντας το μενού Αρχείο/Νέο/Σχέδιο και στη συνέχεια χρησιμοποιώντας τα εργαλεία Σχεδίασης που είναι ενσωματωμένα στο JFY Core.
