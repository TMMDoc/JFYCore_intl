Ohjelmiston asentaminen

Asennuksen valmistelu

Järjestelmävaatimukset

  • Jokin seuraavista käyttöjärjestelmistä. Huomaa, että Windows 7 -käyttöjärjestelmään tulee olla asennettu Service Pack 1 päivitys. Muutoin JFY Core ei toimi Windows 7 -käyttöjärjestelmässä.

    • Windows 7 SP1, 64-bittinen (sovellus tukee Windows 7 SP1 -päivitysversiota Extended Security Updates -ohjelman ollessa asennettuna)

    • Windows 8.1, 64-bittinen

    • Windows 10, 64-bittinen

    • Windows 11, 64-bittinen

  • 8 GB RAM (suositus: 16 GB)

  • Näytön resoluutio 1024 x 768 (suositus: 1920 x 1080)

  • OpenGL 4.0 -rajapintaa tukeva näytönohjain (suositus: Nvidian erillinen grafiikkasuoritin)

  • 3-painikkeinen hiiri tai pallohiiri

Tarvittava ohjelmisto

JFY Core-sovelluksen käyttö vaatii seuraavia ohjelmistoja, ja sovelluksen asennusohjelma asentaa nämä ohjelmistot automaattisesti, jos ne puuttuvat järjestelmästä:

  • Visual C++ 2005 SP1 Redistributable (x86)

  • Visual C++ 2010 SP1 Redistributable (x86)

  • Visual C++ 2013 Redistributable (x64) – vain 64-bittiset alustat

  • Visual C++ 2019 Redistributable (x64) – vain 64-bittiset alustat

  • Microsoft .Net Framework 4.7.2

  • Microsoft .Net 6.0.1

Ohjelmiston asentaminen

Suorita ohjelma JFY CoreSetup.exe (tarkka tiedostonimi voi poiketa tästä, ja saattaa sisältää versionumeron tai päivämäärän). Jos tarvittavat ohjelmistot (kuten .NET Framework) eivät ole valmiiksi asennettuina, asennus voi kestää jopa 15 minuuttia. Jos tarvittavat ohjelmistot ovat valmiiksi asennettuina, asennuksen pitäisi kestää noin 1–2 minuuttia.

Asenna JFY Core noudattamalla näyttöön ilmestyviä ohjeita. Asennusprosessi sijoittaa JFY Core-sovelluksen pikakuvakkeen työpöydällesi ja aloitusvalikkoon. Voit myös käynnistää sovelluksen suorittamalla JFY Core.exe-ohjelman suoraan.

JFY Core-sovelluksen käynnistäminen

Kun käynnistät JFY Core-sovelluksen ensimmäisen kerran, näyttöön avautuu kielten valikko, josta voit valita JFY Core-sovelluksen käyttämän kielen. (Kielten valikoima voi vaihdella eri asennuskohtaisesti).

Tässä valitsemasi kieli on tämän jälkeen käytössäJFY Core-sovelluksessa. Valintaa voi myöhemmin vaihtaa napsauttamalla painiketta Language komentopalkissa. (Tämä kuvake ilmestyy näyttöön käynnistäessäsi JFY Core-sovelluksen ilman, että sinulla on ollut osia avattuna.)
