JFY Bend

JFY Bend

Μια επίπεδη μεταλλική πλάκα μπορεί να καμφθεί στο τελικό της σχήμα χρησιμοποιώντας μια καμπτική πρέσα. Εδώ παρατίθενται μερικές από τις εργασίες που εκτελεί το JFY Bend, όταν χρησιμοποιείται για τον προγραμματισμό αυτών των μηχανημάτων:

  • Υπολογίστε μια εφικτή και βέλτιστη αλληλουχία κάμψης που αποφεύγει τις συγκρούσεις και ελαχιστοποιεί τις περιστροφές ή τις αναστροφές των τεμαχίων.

  • Συνθέστε τους σταθμούς εργαλείων κάμψης από τη διαθέσιμη συλλογή εργαλείων κάμψης, χρησιμοποιώντας στιγείς, μήτρες, προσαρμογείς και επεκτάσεις, όπως απαιτείται.

  • Υπολογίστε τις θέσεις ανατολής όπου μπορούν να τοποθετηθούν οι πίσω αναστολείς για να ευθυγραμμίσουν το τεμάχιο για την κάμψη.

  • Διαμορφώστε ειδικό εξοπλισμό όπως η μέτρηση γωνίας ή τα βάσεις απόθεσης κάμψης που μπορεί να διαθέτει η καμπτική πρέσα.

  • Δημιουργήστε ένα πρόγραμμα NC που μπορεί να μεταδοθεί στην καμπτική πρέσα για επεξεργασία.

  • Δημιουργήστε αναφορές (όπως Φύλλα ρύθμισης) για τη διαδικασία κάμψης.

Το JFY Bend ενσωματώνει αλγορίθμους και ευρετικές μεθόδους για τον υπολογισμό της λύσης κάμψης αυτόματα και στις περισσότερες περιπτώσεις απαιτείται συνήθως ελάχιστη ή καθόλου παρέμβαση. Ωστόσο, το JFY Bend παρέχει επίσης ένα πλούσιο σύνολο εργαλείων και λειτουργιών για τη διαδραστική τροποποίηση της αλληλουχίας κάμψης, των ρυθμίσεων εργαλείων, των θέσεων των αναστολέων και κάθε παραμέτρου της διαδικασίας κάμψης.

Καθώς πραγματοποιούνται αυτές οι αλλαγές, διάφοροι έλεγχοι (συμπεριλαμβανομένου ενός ολοκληρωμένου ελέγχου συγκρούσεων και ελέγχων σκοπιμότητας) διεξάγονται σε πραγματικό χρόνο και μπορείτε πάντοτε να προβάλλετε τα αποτελέσματα των τροποποιήσεών σας καθώς εργάζεστε, χωρίς να χρειάζεται να ζητήσετε επανυπολογισμό.

Έναρξη

Ακολουθούν ορισμένες πληροφορίες για να ξεκινήσετε με το JFY Bend:

  • Όλες οι ενότητες JFY Bend χρησιμοποιούν από κοινού ένα μοντέλο διάδρασης χρήστη που χαρακτηρίζεται ως διεπαφή Clarity. Διαβάστε την ενότητα για τη διεπαφή Clarity όταν ξεκινάτε με το JFY Bend.

  • Μπορείτε να εγκαταστήσετε καμπτικές πρέσες από τη Βάση δεδομένων μηχανήματος και μπορείτε να τις διαμορφώσετε έτσι ώστε να ταιριάζουν με τα πραγματικά μηχανήματα που είναι διαθέσιμα στο κατάστημά σας.

  • Μπορείτε να χρησιμοποιήσετε τη βάση δεδομένων εργαλείων κάμψης για να εγκαταστήσετε καταλόγους εργαλείων, να εισαγάγετε προσαρμοσμένα εργαλεία και να διαμορφώσετε τη συλλογή εργαλείων κάμψης ώστε να ταιριάζουν με αυτά που έχετε πραγματικά στο κατάστημά σας.

  • Το σημείο εκκίνησης για όλες τις μονάδες CAM είναι τα δεδομένα CAD και μπορείτε να αποκτήσετε αυτά τα αρχεία CAD εισάγοντας δεδομένα 2D ή 3D ή ακόμα και σχεδιάζοντάς τα χρησιμοποιώντας το ενσωματωμένα εργαλεία σχεδίασης του JFY Bend.

Μόλις υπολογίσετε μια λύση κάμψης:

  • Μπορείτε να χρησιμοποιήσετε τους διάφορους Πίνακες επεξεργαστή για να κάνετε απευθείας κλικ και να επεξεργαστείτε χαρακτηριστικά όπως η ρύθμιση του εργαλείου ή η αναστολή.

  • Ο Πλοηγός κάμψης μπορεί να χρησιμοποιηθεί ως πίνακας ελέγχου για γρήγορη προβολή και επεξεργασία της κατάστασης ολόκληρης της λειτουργίας κάμψης. Κάνοντας κλικ σε ένα κελί προειδοποίησης ή σφάλματος εδώ μεταβαίνετε στην πηγή του προβλήματος, την οποία μπορείτε στη συνέχεια να διορθώσετε διαδραστικά.

  • Η ενότητα ]Για προχωρημένουςxref:Advanced/Index#[ περιγράφει πρόσθετες λειτουργίες της τεχνολογίας κάμψης συμπεριλαμβανομένων του επανυπολογισμού της λύσης με τροποποιημένες ρυθμίσεις και της επεξεργασίας παρτίδας.
