Reports

Reports

นอกเหนือจากการสร้างโปรแกรม NC JFY Bend ยังสามารถสร้าง_รายงาน_ หลากหลายประเภทสำหรับชิ้นงาน ทั้งนี้ขึ้นอยู่กับโมดูลที่ติดตั้ง ตัวอย่างเช่น เมื่อทำงานกับเทคโนโลยีการพับ จะเป็นประโยชน์ในการสร้าง setup sheet ที่ระบุเครื่องมือที่จะติดตั้ง และแสดงรายละเอียดของ ลำดับการพับ

Report Designer ใน JFY Bend ช่วยคุณออกแบบเนื้อหาและสไตล์ของ รายงาน เมื่อคุณออกแบบรายงานเสร็จแล้ว คุณสามารถส่งออกเป็น PDF หรือ HTML[1] รูปแบบต่างๆ นี่คือตัวอย่างหน้าตาของ report designer เมื่อถูกเปิดขึ้น:

การใช้งานรายงาน

คุณสามารถเปิดรายงานเพื่อแก้ไขได้ โดยเลือกเมนู File/Open และเลือก ไฟล์แม่แบบรายงาน (นามสกุล .ftrep) คุณยังสามารถ หาไฟล์ตัวอย่างแม่แบบรายงานได้ในโฟลเดอร์ Samples ของการติดตั้ง JFY Bend

ก่อนที่คุณจะเปิดแม่แบบรายงานเพื่อใช้งาน คุณควรมีชิ้นงานเปิดอยู่ก่อน และชิ้นงานนั้นต้องถูกประมวลผลเสร็จสิ้นแล้ว (คุณควรสร้าง ข้อมูลเทคโนโลยี Bend/Cut/Fold ที่เหมาะสมไว้ก่อน) เหตุผลคือ report designer ทำงานด้วย live data ซึ่งจะแสดงข้อมูลจริงจากชิ้นงานทันทีที่คุณออกแบบรายงาน เพื่อให้คุณเห็นผลลัพธ์การพิมพ์จริงล่วงหน้า

คุณยังสามารถเริ่มรายงานเปล่าได้โดยเลือกเมนู File/New/Report Template จากนั้นคุณสามารถใส่_องค์ประกอบของรายงาน_ (ฟิลด์ข้อความ, รูปภาพ, labels ฯลฯ) แล้วบันทึกได้

เมื่อคุณเปิดแม่แบบรายงานแล้ว คุณสามารถใช้คำสั่ง File/Export to PDF หรือ File/Export to HTML เพื่อ_เรนเดอร์_รายงานออกมาเป็น PDF หรือ HTML รายงานจะถูกพิมพ์โดยใช้ข้อมูลจากชิ้นงานที่เปิดและทำงานอยู่ในปัจจุบัน คุณยังสามารถสลับการมองระหว่าง live data view (ค่าเริ่มต้น) กับ design view ที่แสดง expression พื้นฐานที่ใช้เติมข้อมูลใน ฟิลด์ต่างๆ ของรายงาน

รายงานที่พิมพ์อัตโนมัติ:

คุณสามารถตั้งค่า JFY Bend ให้ระบบพิมพ์รายงานการพับอัตโนมัติ ทุกครั้งที่มีการสร้างคำตอบการพับ (เหมือนกับการสร้างโปรแกรม NC อัตโนมัติ เมื่อคำนวณเสร็จสิ้น) ใช้หน้าการตั้งค่า outputs ของโมดูลเทคโนโลยีการพับ เพื่อกำหนดการทำงานนี้

ในตัวอย่างด้านบน การสร้างรายงานอัตโนมัติถูกเปิดใช้งาน แม่แบบ รายงานถูกตั้งไว้ที่ lib:Report/bend.ftrep ซึ่งเป็นแม่แบบรายงาน_เริ่มต้น_ สำหรับ bend CAM (ไฟล์นี้ถูกติดตั้ง JFY Bend ทุกครั้ง) หากคุณสร้างแม่แบบรายงานใหม่โดยใช้ report designer คุณสามารถ ระบุให้ระบบใช้แม่แบบนั้นแทนได้

การตั้งค่ายังอนุญาตให้คุณกำหนดโฟลเดอร์ปลายทางสำหรับการสร้างรายงาน และรูปแบบ (HTML หรือ PDF) ที่จะบันทึก

1. ไฟล์ HTML ที่สร้างโดย JFY Bend report writer เป็นไฟล์ที่ self-contained ไฟล์เหล่านี้จะรวมรูปภาพที่จำเป็นทั้งหมด, style sheet และ resource อื่นๆ ที่ฝังอยู่ภายในไฟล์ HTML ดังนั้นจึงสามารถส่งต่อหรือนำไปเก็บถาวรได้ง่าย โดยไม่ต้องพึ่งพา resource ภายนอกใดๆ
