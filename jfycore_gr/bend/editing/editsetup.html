Επεξεργασία ρυθμίσεων κάμψης

Όταν επεξεργάζεστε μια βάση κάμψης, μπορείτε να κάνετε κλικ στο κουμπί πλοήγησης Setup στον πίνακα βάσης εργαλείου για να επεξεργαστείτε τις πλήρεις ρυθμίσεις κάμψης. Ο πίνακας ρυθμίσεων κάμψης προβάλλεται παράλληλα και παρέχει τη δυνατότητα εκτέλεσης ορισμένων διαδικασιών.

  • Μπορείτε να επιλέξετε Save τις ρυθμίσεις για ένα αρχείο ρυθμίσεων κάμψης (αυτά είναι αρχεία με την επέκταση .bsetup). Σε μεταγενέστερο στάδιο μπορείτε να χρησιμοποιήσετε την επιλογή Load…​ για να φορτώστε αυτές τις ρυθμίσεις για χρήση με διαφορετικό τεμάχιο. Αυτός είναι ένας απλός, χειροκίνητος τρόπος χρήσης των ίδιων ρυθμίσεων για πολλά τεμάχια.

  • Το κουμπί Merge Setups προβάλλεται όταν υπάρχουν πολλές ρυθμίσεις στο τεμάχιο και συγχωνεύει αυτές τις πολλαπλές ρυθμίσεις σε μία, εάν υπάρχει επαρκής χώρος στο μηχάνημα για όλα τα setups[1].

  • Το κουμπί Center Setup κουμπί προβάλλεται όταν οι σταθμοί σε αυτές τις ρυθμίσεις δεν είναι κεντραρισμένοι στη διαθέσιμη επιφάνεια τραπεζιού του μηχανήματος.

  • Τα κουμπιά Next και Prev μπορούν να χρησιμοποιηθούν για εναλλαγή μεταξύ διαφορετικών ρυθμίσεων σε αυτό το τεμάχιο (εάν υπάρχουν πολλαπλές ρυθμίσεις).

  • Κάνοντας κλικ στο κουμπί πλοήγησης Mounts κατεβαίνετε κατά ένα επίπεδο για την επεξεργασία της πρώτης βάσης εργαλείου σε αυτές τις ρυθμίσεις (είναι το αντίθετο του κουμπιού πλοήγησης Setup στον πίνακα βάσης εργαλείου).

1. Εάν οι ρυθμίσεις χρησιμοποιούν εργαλεία διαφορετικού ύψους, δημιουργούνται συχνά συγκρούσεις μεταξύ των στιγέων και των μητρών, γεγονός που αναφέρεται στον πλοηγό κάμψης.
