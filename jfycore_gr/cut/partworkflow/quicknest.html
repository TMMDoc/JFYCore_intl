Quick-Nest (Γρήγορη κατανομή)

Η λειτουργία Quick-Nest χρησιμοποιείται για τη δημιουργία μιας κατανεμημένης μεταλλικής πλάκας με μόνο ένα τεμάχιο. Στη συνέχεια, μπορεί να δημιουργηθεί ένα πρόγραμμα NC και ένα φύλλο ρυθμίσεων για αυτήν την αναφορά. Ένας τρόπος για την εκτέλεση μιας γρήγορης κατανομής συνίσταται στη χρήση του πίνακα Ροή εργασιών.

  • Κάνετε κλικ στο W για να ανοίξετε τον πίνακα ροής εργασιών.

  • Χρησιμοποιήστε το CtrlΚλικ+ στο εικονίδιο Γρήγορη κατανομή για να ανοίξετε το παράθυρο ρυθμίσεων κατανομής, όπου μπορείτε να ρυθμίσετε το μέγεθος της μεταλλικής πλάκας και την απόσταση μεταξύ των τεμαχίων.

  • Κάνετε κλικ στο εικονίδιο Γρήγορη κατανομή για να δημιουργήσετε μια μεταλλική πλάκα με το μεμονωμένο τεμάχιο ή το ShiftΚλικ+ για να δημιουργήσετε μια μεταλλική πλάκα με ένα μόνο τεμάχιο.

  • Μετά τη δημιουργία της διάταξης, μπορείτε να συνεχίσετε κάνοντας κλικ στο εικονίδιο Μετεπεξεργασία διάταξης για τον υπολογισμό του προγράμματος NC και της αναφοράς για τη διάταξη.

Στην παρακείμενη εικόνα παρουσιάζεται η αναφορά που έχει δημιουργηθεί για μια κατανεμημένη μεταλλική πλάκα.

Μπορείτε επίσης να εκτελέσετε όλα αυτά τα βήματα αρχίζοντας με ένα τεμάχιο 2D χρησιμοποιώντας αυτές τις συντομεύσεις πληκτρολογίου:

  • Κάνετε κλικ στο L για να δημιουργήσετε εργαλεία λέιζερ για το τεμάχιο.

  • Κάνετε κλικ στο N για να εκτελέσετε μια γρήγορη κατανομή.

  • Κάνετε κλικ στο C για να δημιουργήσετε το πρόγραμμα NC και την αναφορά διάταξης.

Μια πιο ολοκληρωμένη μέθοδος για τη δημιουργία μιας κατανεμημένης μεταλλικής πλάκας με πολλά τεμάχια και για πιο λεπτομερή έλεγχο της διαδικασίας κατανομής συνίσταται στη δημιουργία μιας JFY Cutεργασίας κατανομής.
