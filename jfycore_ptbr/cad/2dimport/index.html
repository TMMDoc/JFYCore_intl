Importação 2D

O JFY Core pode importar desenhos 2D de arquivos DXF ou GEO. Esses desenhos são tipicamente 2D planos - são desdobramentos ou desenvolvimentos de uma peça de chapa 3D, pronta para processamento com um cortador a laser.

Para processar para máquinas a laser, prensas-dobradeiras ou máquinas de dobra oscilante, JFY Core precisa de um desenho da chapa que contém:

  • Um contorno externo fechado

  • Um ou mais contornos internos (furos)

  • Uma ou mais linhas de dobra

Consulte a página de Limpeza 2D para obter mais informações sobre como usar as ferramentas de modo de esboço para limpar um desenho ou para adicionar anotações de linha de dobra. Uma vez que o desenho está pronto, ele pode então ser usado para processamento adicional:

  • Pressione 3 para dobrar o desenho para um modelo 3D (usando as anotações da linha de dobra presentes no desenho).

  • Pressione L para processar o desenho para corte a laser.

  • Pressione B para processar o desenho para dobra em V em uma prensa-dobradeira

  • Pressione F para processar o desenho em um máquina de dobra oscilante

  • Pressione W para abrir o painel Fluxo de trabalho para fazer todo o processamento de uma maneira mais interativa.

Você também pode criar um desenho 2D completamente do zero usando o menu Arquivo / Novo / Desenho e, em seguida, usando as ferramentas de Esboço incorporadas ao JFY Core.
