Back-gauges

Min.thickness to always pull-back (mm) - Esta configuração permite inserir um valor em mm de retirada automática dos encostos traseiro quando a espessura da chapa inserida é atingida.

Clamping strategy - Isso é usado para definir como os encostos traseiros devem parar na peça de trabalho.

As seguintes opções de seleção estão disponíveis:

Default - Os encostos traseiros param para cada dobra do modo considerado melhor pelo software.

Stop + Stop - As superfícies retas de ambos os encostos traseiros param contra a peça de trabalho.

Clamp + Stop - Um encosto traseiro prende a peça de trabalho e o outro encosto traseiro a para com sua superfície reta.

Geometry Adaptation Strategy

Se um arquivo 2D ou 3D for editado no software, os raios de dobra e a dedução de dobra são verificados. Diversas estratégias de processamento são executadas dependendo de quais estratégias foram selecionados para modificar a peça de trabalho nas configurações do Easy Programming. Se a decisão foi definida como interativa, será mostrado o diálogo Adapt Geometry.

When starting with 2D flat

Ask each time - Ao navegar de uma peça 2D para dobrar, usar esta configuração exibirá automaticamente o diálogo Adapt Geometry. Isso dará opções de retenção do tamanho plano, mantendo as dimensões 3D ou não alterando nada, tornando esse recurso muito interativo e conduzido pelo usuário.

Keep 3D dimensions - Esta opção altera o desdobramento, que então deve ser salvo manualmente. Esta estratégia é usada se o espaço em branco ainda não foi cortado.

Keep 2D dimensions - Esta opção significará que o raio de dobra e a dedução de dobra são ajustados ao desdobramento existente; a linha de dobra permanece no mesmo lugar. Isso mudará o comprimento lateral da peça. O desdobramento não é alterado. Esta estratégia é usada se o espaço em branco já tiver sido cortado ao tamanho.

No adjustments - Usar esta opção significará que o raio de dobra e a dedução de dobra permanecem inalterados. A peça é exibida com as dimensões corretas em 3D, mas os raios de dobra mostrados não correspondem aos raios de dobra reais. A dimensão de cada dobra da peça de trabalho pode mudar.

When starting with 3D model

Ask each time - Ao navegar de uma peça 3D para dobrar, usar esta configuração exibirá automaticamente o diálogo Adapt Geometry. Isso dará opções de retenção do tamanho plano, mantendo as dimensões 3D ou não alterando nada, tornando esse recurso muito interativo e conduzido pelo usuário.

Keep 3D dimensions - Esta opção altera o desdobramento, que então deve ser salvo manualmente. Esta estratégia é usada se o espaço em branco ainda não foi cortado.

Keep 2D dimensions - Esta opção significará que o raio de dobra e a dedução de dobra são ajustados ao desdobramento existente; a linha de dobra permanece no mesmo lugar. Isso mudará o comprimento lateral da peça. Esta estratégia é usada se o espaço em branco já tiver sido cortado ao tamanho.

No adjustments - Usar esta opção significará que o raio de dobra e a dedução de dobra permanecem inalterados. A peça é exibida com as dimensões corretas em 3D, mas os raios de dobra mostrados não correspondem aos raios de dobra reais. A dimensão de cada dobra da peça de trabalho pode mudar.

Bend deduction calculation - Esta opção possibilita forçar o método do Fator K a ser usado. Esta configuração pode ser definida nos Padrões da Aplicação ou pode ser ajustada individualmente para cada máquina.

Para máquinas não Trumpf, os fatores K são usados sempre; os cálculos de dobre Trumpf estão disponíveis somente para prensas dobradeiras Trumpf.

BG-Left variant - Dependendo da versão BG utilizada, o programa CN será modificado para representar isso.

BG-Right variant - Dependendo da versão BG utilizada, o programa CN será modificado para representar isso. == Other Settings **Hemming pre-bend angle - Esta opção pode editar o ângulo de pré-dobra do vinco entre 10° a 60°. O padrão é 40°, já que esse é o ângulo de pré-dobra de vincagem recomendado para a maioria dos fabricantes. Está definido para 30° para todas as máquinas Trumpf, em todas as diversas edições.

Bend-guard mode override - Ao configurar isso para cálculo automático (o padrão), Bend calculará um modo de proteção de dobra dobra. No entanto, você pode forçar o modo de proteção de dobra para uma das opções para ele usar somente aquela, independentemente de ser apropriado ou não.

Você também pode substituir essa configuração para cada máquina, configurando-a nos Padrões da Máquina, em vez de Application Settings.

Machine table position - Esta configuração tem três opções para a posição da mesa:

  1. Removida

  2. Para baixo

  3. Para cima

Tool inventory check - Ativar esta opção mostrará os segmentos da ferramenta que podem ser necessários para que a peça de trabalho seja dobrada. Marcadores de inventário insuficientes serão exibidos.

Avoid tool overload if possible - Ativar esta opção assegurará que seja evitada uma sobrecarga da ferramenta. Mesmo que uma ferramenta com um maior prioridade funcionasse na peça, uma ferramenta de prioridade mais baixa será usada para evitar problemas de sobrecarga.

Ignore tool hints from GEO/DXF files - Ativar esta opção ignorará qualquer informação encontrada em um arquivo DXF/GEO em relação ao qual ferramental deve ser usado.
