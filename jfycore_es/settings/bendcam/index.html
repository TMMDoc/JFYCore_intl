CAM de plegado

CAM de plegado

Back-gauges

Min.thickness to always pull-back (mm) - Este ajuste le permite introducir un valor en mm de retirada automática de los topes traseros cuando se alcanza el espesor de chapa introducido.

Clamping strategy - Se utiliza para ajustar cómo deben detenerse los topes traseros en la pieza de trabajo.Las siguientes opciones de selección están disponibles:

Default - Los topes traseros se detienen en cada plegado de la manera que el software considera más adecuada.

Stop + Stop - Las superficies rectas de ambos topes traseros se detienen contra la pieza de trabajo.

Clamp + Stop - Un tope trasero sujeta la pieza de trabajo y el otro tope trasero la detiene con su superficie recta.

Geometry Adaptation Strategy

Si se edita un archivo 2D o 3D en el software, se comprueban los radios de plegado y la reducción de plegado. Se ejecutan diversas estrategias de procesamiento en función de las estrategias seleccionadas para modificar la pieza de trabajo en la configuración de programación fácil. Si la decisión se estableció como interactiva, se mostrará el diálogo Adapt Geometry.

When starting with 2D flat

Ask each time - Al pasar de una pieza 2D al plegado, al utilizar esta configuración se mostrará automáticamente el diálogo Adapt Geometry. Esto ofrecerá opciones para mantener el tamaño del plano, conservar las dimensiones 3D o no cambiar nada, lo que hace que esta función sea muy interactiva y orientada al usuario.

Keep 3D dimensions - Esta opción cambia el desarrollo que luego debe guardarse manualmente. Esta estrategia se utiliza si la pieza bruta aún no se ha cortado.

Keep 2D dimensions - Esta opción significa que el radio de plegado y la reducción de plegado se ajustan al desarrollo existente; la línea de plegado permanece en el mismo lugar. Esto cambiará la longitud lateral de la pieza. El desarrollo no se modifica. Esta estrategia se utiliza si la pieza bruta ya se ha cortado a medida.

No adjustments - Si se utiliza esta opción, el radio de plegado y la reducción de plegado permanecerán sin cambios. La pieza se muestra con las dimensiones correctas en 3D, pero los radios de plegado mostrados no se corresponden con los radios de plegado reales. La dimensión de cada plegado de la pieza de trabajo puede variar.

When starting with 3D model

Ask each time - Al pasar de una pieza 3D al plegado, al utilizar esta configuración se mostrará automáticamente el diálogo Adapt Geometry. Esto ofrecerá opciones para mantener el tamaño del plano, conservar las dimensiones 3D o no cambiar nada, lo que hace que esta función sea muy interactiva y orientada al usuario.

Keep 3D dimensions - Esta opción cambia el desarrollo que luego debe guardarse manualmente. Esta estrategia se utiliza si la pieza bruta aún no se ha cortado.

Keep 2D dimensions - Esta opción significa que el radio de plegado y la reducción de plegado se ajustan al desarrollo existente; la línea de plegado permanece en el mismo lugar. Esto cambiará la longitud lateral de la pieza. Esta estrategia se utiliza si la pieza bruta ya se ha cortado a medida.

No adjustments - Si se utiliza esta opción, el radio de plegado y la reducción de plegado permanecerán sin cambios. La pieza se muestra con las dimensiones correctas en 3D, pero los radios de plegado mostrados no se corresponden con los radios de plegado reales. La dimensión de cada plegado de la pieza de trabajo puede variar.

Bend deduction calculation - Esta opción permite forzar el uso del método del factor K. Este ajuste se puede establecer en los valores predeterminados de la aplicación o se puede ajustar individualmente para cada máquina.

Para máquinas que no sean Trumpf, siempre se utilizan factores K; los cálculos de plegado de Trumpf solo están disponibles para prensas plegadoras Trumpf.

BG-Left variant - Dependiendo de la variante BG utilizada, el programa CN se modificará para reflejarlo.

BG-Right variant - Dependiendo de la variante BG utilizada, el programa CN se modificará para reflejarlo.

Other Settings

Hemming pre-bend angle - Esta opción puede editar el ángulo previo de prensado entre 10° y 60°. El valor predeterminado es 40°, ya que es el ángulo de prensado previo recomendado para la mayoría de los fabricantes. Está ajustado a 30° para todas las máquinas Trumpf, en todas las distintas ediciones.

Bend-guard mode override - Si se configura en cálculo automático (el valor predeterminado), Bend calculará un modo de protección de plegado para cada plegado. Sin embargo, puede forzar el modo de protección de plegado a una de las opciones para que solo utilice esa, independientemente de si es adecuada o no.

También puede anular esta configuración para cada máquina, estableciéndola en los valores predeterminados de la máquina, en lugar de Application Settings.

Machine table position - Este ajuste tiene tres opciones para la posición de la mesa:

  1. Retirada

  2. Abajo

  3. Arriba

Tool inventory check - Al habilitar esta opción, se mostrarán segmentos de útil que pueden ser necesarios para plegar la pieza de trabajo. Se mostrarán los marcadores de inventario insuficientes.

Avoid tool overload if possible - Al activar esta opción, se evitará la sobrecarga del útil. Incluso si un útil con mayor prioridad funcionara en la pieza, se utilizará un útil con menor prioridad para evitar problemas de sobrecarga.

Ignore tool hints from GEO/DXF files - Al habilitar esta opción, se ignorará cualquier información encontrada en un archivo DXF/GEO sobre qué útiles se deben utilizar.
