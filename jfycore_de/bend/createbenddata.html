Biegeparameter erstellen

Biegeparameter erstellen

Mit dem Biegemodul können Sie Biegetechnologiedaten für ein Blechteil erstellen. Dies wird auch als das Rüsten des Teils für das Biegen bezeichnet. Nehmen wir an, Sie haben bereits eine Maschine installiert, den Werkzeugbestand konfiguriert und ein CAD-Teil (2D oder 3D) importiert:

Umschalten auf Biege-CAM

Jetzt kann das Teil durch einfaches Drücken der Taste B für das Biegen gerüstet werden. Das Teil wird mit der Standardbiegemaschine gerüstet, bei der es sich um die Abkantpresse handelt, die Sie zuletzt verwendet haben. Es werden eine Werkzeugreihenfolge, eine Biege-Aufspannung und Anschlagpositionen berechnet, und Sie sollten ein Ergebnis wie dieses sehen:

Liegen keine Fehler vor, wird das NC-Programm (und ggf. der Einrichteplan) für das Teil ebenfalls generiert und in dem Ausgabeordner gespeichert, der für diese Maschine eingerichtet ist. Hier ist ein Abschnitt aus einer typischen Teile-Einrichtung. sheet[1] (der auch als als Biegebericht bezeichnet wird).

Siehe auch den Fensterbereich Workflow, auf den Sie durch Drücken der Taste W zugreifen können. Dies bietet mehr Kontrolle über die Erstellung von Biegetechnologiedaten und Optionen zum Routen des Teils durch Laser-Cam oder Schachteln.

Sonstige Vorgänge

Hier finden Sie eine kurze Zusammenfassung einiger Vorgänge, die Sie durchführen können, sobald ein Teil zum Biegen gerüstet wurde.

  • Wenn Sie auf die Spacebar klicken, wird eine Biegesimulation gestartet. Sie können auch die Simulationssteuerelemente im Biegenavigator verwenden, um die Simulation zu starten, zu stoppen oder zurückzuspulen:

  • Wenn Warnungen oder Fehler auftreten, erscheinen sie im Biegenavigator, und Sie können sie durch Anklicken der entsprechenden Zellen anzeigen und beheben:

  • Sie können die Biegewerkzeuge (Stempel und Matrizen), die Anschlageinstellungen oder die Winkelmessungseinstellungen bearbeiten, indem Sie einfach in der Simulationsansicht direkt auf das jeweilige Objekt klicken:

  • Sie können die Maschine ändern, indem Sie auf den Maschinennamen auf dem Tab unter dem Teil klicken und eine andere Maschine wählen:

  • Klicken Sie auf das Icon Einstellungen in der Symbolleiste auf der linken Seite, um zusätzliche Einstellungen für die einzelnen Biegungen zu bearbeiten (oder um die Standardeinstellungen zu bearbeiten, die für diese Biegemaschine oder für die gesamte JFY Bend-Anwendung verwendet werden).

  • Klicken Sie auf das Icon Anzeige Symbol in der linken Symbolleiste, um die Ansicht der Maschine zu bearbeiten - Sie können die Anzeige von zusätzlichen Komponenten wie der Matrizenschiene, dem Biegepressbalken oder dem Anschlagfinder-Stützsystem ein- oder ausschalten. Sie können auch verschiedene Komponenten durchscheinend machen.

Wenn ein Biegeprogramm fehlerfrei erstellt wurde, werden automatisch das NC-Programm des Teils und ein Einrichteplan für das Biegen generiert. Sie können die Einstellungsseite Bend Outputs verwenden, um einzustellen, ob das NC-Programm des Teils und der Einrichteplan automatisch generiert werden, und auch um das Ziel für diese Ausgaben sowie das Format der Berichte zu konfigurieren. Wenn ein Programm generiert wurde, ist die Schaltfläche NC-Programm in der linken Symbolleiste ausgegraut:

Wenn Sie Änderungen am Teil vornehmen, wird diese Schaltfläche wieder aktiviert, und Sie können das NC-Programm neu generieren, indem Sie erneut auf dieses Icon klicken (oder die Taste C drücken).

1. Sie können den Berichts-Editor verwenden, um diesen Einrichteplan anzupassen.
