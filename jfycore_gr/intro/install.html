Εγκατάσταση του λογισμικού

Προετοιμασία για εγκατάσταση

Απαιτήσεις συστήματος

  • Ένα από τα ακόλουθα λειτουργικά συστήματα. Λάβετε υπόψη ότι τα Windows 7 απαιτούν την εγκατάσταση του Service Pack 1, διαφορετικά το JFY Core δεν μπορεί να λειτουργήσει στα Windows 7.

    • Windows 7 SP1, 64-bit (τα Windows 7 SP1 υποστηρίζονται με την εγκατάσταση εκτενών ενημερώσεων ασφαλείας)

    • Windows 8,1, 64-bit

    • Windows 10, 64-bit

    • Windows 11, 64-bit

  • 8 GB RAM (συνιστάται 16 GB)

  • Ανάλυση οθόνης 1024 x 768 (συνιστάται 1920 x 1080)

  • Κάρτα γραφικών με υποστήριξη OpenGL 4.0 (συνιστάται η GPU discrete της Nvidia)

  • Ποντίκι με 3 κουμπιά ή trackball

Προαπαιτούμενο λογισμικό

Απαιτείται το παρακάτω προαπαιτούμενο λογισμικό, το οποίο εγκαθίσταται αυτόματα από το πρόγραμμα εγκατάστασης JFY Core, εάν δεν βρεθεί:

  • Visual C++ 2005 SP1 Redistributable (x86)

  • Visual C++ 2010 SP1 Redistributable (x86)

  • Visual C++ 2013 Redistributable (x64) – μόνο σε πλατφόρμες 64 bit

  • Visual C++ 2019 Redistributable (x64) – μόνο σε πλατφόρμες 64 bit

  • Microsoft .NET Framework 4.7.2

  • Microsoft .Net 6.0.1

Εγκατάσταση του λογισμικού

Εκτελέστε το αρχείο JFY CoreSetup.exe (το ακριβές όνομα αρχείου μπορεί να διαφέρει και να περιλαμβάνει τον αριθμό ή την ημερομηνία αναθεώρησης). Εάν τα προαπαιτούμενα (όπως το .Net Framework) δεν έχουν ήδη εγκατασταθεί, η εγκατάσταση μπορεί να διαρκέσει έως 15 λεπτά μέχρι να ολοκληρωθεί. Εάν τα προαπαιτούμενα έχουν ήδη εγκατασταθεί, η εγκατάσταση θα διαρκέσει περίπου 1 ή 2 λεπτά.

Ακολουθήστε τις οδηγίες για την εγκατάσταση του JFY Core. Αυτή η διαδικασία εγκατάστασης θα τοποθετήσει μια συντόμευση για την εφαρμογή JFY Core στην επιφάνεια εργασίας και στο μενού Έναρξη. Μπορείτε επίσης να ξεκινήσετε την εφαρμογή εκτελώντας απευθείας το πρόγραμμα JFY Core.exe.

Εκτέλεση του JFY Core

Όταν εκτελείτε το JFY Core για πρώτη φορά, εμφανίζεται ένας επιλογέας γλώσσας και μπορείτε επιλέξτε τη γλώσσα που χρησιμοποιείται για το JFY Core. (Ανάλογα με την εγκατάστασή σας, μπορεί να δείτε ένα διαφορετικό σύνολο γλωσσών εδώ).

Η γλώσσα που επιλέγετε εδώ χρησιμοποιείται για το JFY Core και μπορεί στη συνέχεια να αλλάξει με κλικ στο Γλώσσα στη γραμμή εντολών. (Αυτό το εικονίδιο εμφανίζεται όταν εκκινείτε το JFY Core και όταν δεν έχετε ανοίξει κανένα τεμάχιο).
